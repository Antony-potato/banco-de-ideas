📱 Contestadora automática para WhatsApp
19 may 2026 ⏸ Standby

Sistema de contestadora de mensajes donde pueda decirle «respóndele a tal persona» y envíe un mensaje por WhatsApp automáticamente.

💡
Flujo: «Hermes, respóndele a María que llego en 15 min» → el sistema envía el mensaje sin que toques el teléfono.

Funcionalidades

  • Identificar contacto por nombre o número
  • Enviar mensaje de texto automáticamente
  • Integración con voz para dictar el mensaje
  • Respuestas rápidas predefinidas
  • Historial de mensajes enviados

Opciones técnicas investigadas

A · WhatsApp Cloud API (oficial)

API oficial de Meta, gratuita hasta 1,000 conversaciones/mes.

Confiable Soportado por Meta Requiere verificación
B · WhatsApp Web automation

selenium/playwright para automatizar WhatsApp Web. Más flexible pero con riesgo de baneo.

Flexible Sin verificación Menos estable
C · Servicios de terceros

Twilio (pagado pero fácil), CallMeBot (gratis pero limitado), wenbot open-source.

Fácil integración Costo variable
