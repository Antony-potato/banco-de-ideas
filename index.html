Contestadora automatica para WhatsApp
19 may 2026 Standby

Sistema de contestadora de mensajes donde pueda decirle «respondelé a tal persona» y envie un mensaje por WhatsApp automaticamente.

Flujo: «Hermes, respondelé a María que llego en 15 min» el sistema envía el mensaje sin que toques el teléfono.

Funcionalidades

  • Identificar contacto por nombre o número
  • Enviar mensaje de texto automaticamente
  • Integración con voz para dictar el mensaje
  • Respuestas rápidas predefinidas
  • Historial de mensajes enviados

Twilio (pagado pero fácil), CallMeBot (gratis pero limitado), wenbot open-source.

Monitoreo de camaras en múltiples locaciones
20 may 2026 Nuevo

Sistema que monitoree cámaras de seguridad Tapo y Steren distribuidas en varias ciudades y envié notificaciones automáticas si alguna cámara se apaga o pierde conexión. Integración con Zoho Cliq y Telegram.

Flujo: Cada 5-10 min se revisa el estado de todas las cámaras vía APIs cloud. Si alguna cae, te avisa por Telegram y Zoho Cliq.

Funcionalidades

  • Monitoreo remoto de cámaras Tapo (API cloud)
  • Monitoreo de cámaras Steren (API Tuya Cloud)
  • Detección de estado offline/apagado en tiempo real
  • Notificaciones por Zoho Cliq y Telegram
  • Dashboard con estado de todas las cámaras
  • Historial de eventos (caidás y recuperaciones)
A · Tapo Cloud API (pytapo)

Se autentica con email/contraseña de la app Tapo. Consulta estado de todas las cámaras remotamente sin acceder a la red local.

Sin configuración de red API no-oficial Python
B · Steren Home / Tuya Cloud

Steren Home probablemente usa plataforma Tuya. Se necesita cuenta developer en developer.tuya.com con Access ID y Secret.

API oficial Requiere setup Por investigar

Zoho Cliq incoming webhook + Telegram (ya conectado). Notificaciones multi-canal.
