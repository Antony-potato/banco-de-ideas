### 1. Microservicios: Clientes App (Docker + Kubernetes + Helm)
**Repos:** `clientes-back` | `clientes-front` | `clientes-helm`

**Descripción:** Arquitectura de microservicios completa con CI/CD para Docker Hub y despliegue en Kubernetes con Helm charts.

**Stack:**
- **Backend:** Node.js + Express + MySQL (REST API)
- **Frontend:** HTML/CSS/JS vanilla servido por nginx con reverse-proxy
- **Infraestructura:** Docker, Docker Compose, Kubernetes, Helm charts
- **CI/CD:** GitHub Actions para publicación automática en Docker Hub
- **Base de datos:** MySQL con PVC persistente en K8s

**Componentes:**
- API REST con healthchecks, connection pooling y retry logic
- Frontend con nginx como reverse-proxy (evita CORS interno en K8s)
- Helm chart completo: MySQL secret, ConfigMap, PVC, Deployments, Services
- Despliegue local con Minikube

**Lo que demuestra:** Conocimiento de DevOps, contenedores, orquestación, CI/CD, arquitectura distribuida
